# Advanced Features

Explore the advanced capabilities of the Obsidian Graph Plugin.

## Link Detection

The plugin automatically detects various link formats:

- **Standard markdown links**: `[text](./path)`
- **Wiki-style links**: `[[page-name]]`
- **Relative paths**: `../sibling/page`

## Node Sizing

Nodes are automatically sized based on their connection count. Documents with more links appear larger in the graph, making it easy to identify central topics.

## Group Coloring

Documents in the same directory share visual grouping, helping you understand the structure of your documentation at a glance.
