# Getting Started

This example site demonstrates the **Obsidian Graph Plugin** for Docusaurus. The plugin automatically scans your docs directory, extracts links between documents, and generates an interactive knowledge graph visualization.

Visit the [Graph page](/graph) to see your documentation connections visualized in an Obsidian-style graph.

## Features

- **Automatic link extraction** - The plugin scans all markdown files and extracts internal links
- **Interactive visualization** - Pan, zoom, and explore your knowledge graph
- **Click to navigate** - Double-click any node to navigate to that document
